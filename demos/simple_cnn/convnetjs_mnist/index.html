Simple CNN MNIST demo from YuFu EDU

Description

This is a simple demonstration showing The operation of a simple Deep learning network. The training is done in this framework, with two layer. After the first layer the, thanks to the max-pooling, the number of pixels is decreased to half. This operation can not be done efficiently on a CNN, since this architectures executes all the operations on all the cells.
